# Artha Digital - Creative Landing Page

A premium, responsive landing page for Artha Digital, built with modern web technologies including Bootstrap, jQuery, and custom CSS animations.

## Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern UI/UX**: Glassmorphism effects, parallax scrolling, and smooth transitions.
- **Interactive Elements**: Owl Carousel, Magnific Popup gallery, and Particle.js backgrounds.
- **Form Integration**: PHP-backed contact and subscription forms (Ready for Mailchimp, GetResponse, etc).

## Technologies Used
- HTML5 & CSS3
- Bootstrap 4/5
- jQuery
- Animate.css
- Particles.js
- Owl Carousel
- Font Awesome

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/ChaithanyaKethireddy/arthadigital.git
   ```
2. Open `index.html` in your browser to view the site.

## Deployment
This project is deployment-ready for static hosting (GitHub Pages, Netlify, Vercel) or traditional PHP hosting.
- **Main file**: `index.html`
- **Configuration**: Contact form settings can be found in `form-data/config.php`.
